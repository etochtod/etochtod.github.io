Система управління навчанням

(прототип, поки що тільки доказ можливості зробити)

Для сайтобудування потрібні три мови: для фронтенда, бекенда, та мова конкретної бази даних. І цю трійцю множимо на два: знадобляться фреймворки для бека і фронта, та ORM для бази даних.

Пітон - краще почати з цієї мови:

  • Легко вивчити.
  • Привчає до чистого кодування.
  • Найкращі бібліотеки.
  • Інші мови після пітона легко вчаться.
  • Корисна мова за межами сайтобудування. Це чудовий прикладний інструмент.
    Завдання виконуються одне за одним.
    Виконане завдання розблокує наступне.

    Відкриті завдання:
  1. Змінні
  2. Об'єкти
  3. Викликання методів
  4. Фініш

При виконанні завдань бажано робити багато помилок. Чим більше помилок, тим краще. Це активує пам'ять.

Чесно кажучи, завдання вивчити пітона автоперевірки не ставлять. Щось запам'ятається, щось забудеться - не так важливо. Задача - отримати загальне уявлення. Після пари десятків завдань по пітону в браузері, буде запропоновано встановити повноцінний python на компьютері. Далі до нього приєднається сервер nginX. Далі додасться... коротше, в кінці буде сайт, який працює на локальному компьютері. Залишиться відкрити доступ в інтернет або задеплоїти на якийсь хостінг.

Зовсім чесно кажучи, і завдання писати з нуля сайти також немає. Справа в тому, що майже весь код вже написаний до нас. Кодеру залишається чужу працю зклеювати/налаштовувати. Для цього потрібна певна підготовка, загальне розуміння як працює кухня сайтобудування. І коли таке розуміння є, далі можна з готових великих блоків швидко робити сайти просто додаючи кілька рядків налаштувань в той чи інший блок. Ось і все програмування. Воно швидке і ефективне, покриває 99% потреб по виготовленню сайтів.

Виключенням з такого блочного програмування є унікальні проекти для яких ще не написаний код. Ідея повинна бути амбіційною. Чим більший замах, тим краще. Чи ідея грандіозна? Чи вона має наміром захопити Світ? Тоді це правильний проект. Але для такого проекта також потрібне загальне уявлення про кухню сайтобудування.
