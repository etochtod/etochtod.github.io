Завдання 1:

Вивести на екран за допомогою змінних name та email текст:

"Привіт, Alex. Твоя пошта alex@company.com"

Використати синтаксис з f.


Змінні. Загальні відомості.

  • В пітоні змінні чутливі до регістру (A відрізняється від a).
  • Декларувати перед використанням не треба (перше присвоювання і є декларацією).
  • Типізація динамічна (вказувати тип змінної не треба).
# integer
a = 5

# float
b = 5.1

# string
c = "text"

# boolean - булеве значення (True або False).
d = True

# list - список (або масив в деяких мовах).
e = [10, 20, 15]

# tuple - кортеж. Як list, тільки його не можна міняти (але він швидкіший та компактніший).
f = (10, 20, 15)

# set - множина (елементи не можуть повторюватися).
g = {10, 20, 15}

# dictionary - пари "ключ":значення.
h = {
    "name": "Alex",
    "age": 27
}

Виведення на екран

На екран виводить команда print()

print(a)
# 5

print("значення b дорівнює: " + b)
# значення b дорівнює: 5.1

print(f"значення a дорівнює {a}, значення b дорівнює {b}")
# значення a дорівнює 5, значення b дорівнює 5.1

Лапки можуть бути як одинарні так і подвійні. Якщо перед лапками стоїть f, то можна вставляти всередину текста фігурні дужки зі змінними. У фігурних дужках можуть бути як змінні так і вирази. Наприклад, {a + b}.
