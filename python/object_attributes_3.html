Задана змінна s. Запустити RUN. Подивитися список атрибутів s. Треба знайти серед них метод для переведення тексту у великі літери, і поміняти код всередині print(...), щоб на екран вивелося значення s великими літерами.


Атрибути об'єктів

В пітоні все є об'єктом або вказівником на об'єкт. Наприклад, 5 є об'єктом. "Hello world!" також є об'єктом.

Об'єкти мають атрибути. Це змінні, які напряму видимі тільки всередині об'єкта. Щоб добратися ззовні до такої змінної треба спочатку вказати назву об'єкта, потім поставити крапку, потім вказати назву змінної (object.attribute).

Оскільки в пітоні все є об'єктом, то замість object може бути все що завгодно, включно з якимись розрахунками (5+7), бо результат розрахунків також є об'єктом.

Подивитися на список атрибутів об'єкта можна командою print(dir(object)). Натиснить RUN щоб побачити атрибути об'єкта "Hello world!".


Дані та методи

Атрибути умовно можна поділити на дані та методи.

               об'єкт
              ↗      ↖
атрибути-дані          атрибути-методи

Дані - це якась інформація. Методи - це код, який цю інформацію обробляє. Команда dir не показує який атрибут є чим, але для виконання методів після назви атрибута додаються дужки. В дужках можуть бути аргументи (змінні), які передаються методу.

s.__doc__
# атрибут-дані. Текст, коротка документація по об'єкту.

s.__dir__()
# атрибут-метод. Команда dir(s) є обгорткою для виконання цього метода. Вона виконує s.__dir__() всередині себе.

s.__add__(s1)
# коли ми складаємо два рядки (s+s1), то це обгортка для цього метода.

s.lower()
# переводить текст у маленькі літери.

Деякі атрибути мають подвійне підкреслення з боків. Це так звані магічні атрибути. Традиційно їх не використовують прямо. Їх використовують опосередковано, викликаючи інші команди.
