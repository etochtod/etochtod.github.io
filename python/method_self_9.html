Функції, методи та self

Блок кода можна прив'язати як атрибут до класа чи інстанса. Якщо ми прив'язали блок кода до класа, то класу він належить прямо (записаний в __dict__), а всі інстанси отримають цей блок кода за наслідуванням (посилання).

Виконання власного кода відрізняється від виконання кода за наслідуванням. В першому випадку блок кода викликається як функція, а в другому - як метод. Якщо код не належить прямо інстансу, а є наслідуванням від класа, то першим аргументом, який автоматично передасться блоку кода, буде посилання на інстанс. Цей параметр не видно, але він буде переданий. В цьому різниця між методом та функцією.

# створимо клас
class Users:
    name = "NoName"

# створимо його інстанс
a = Users()
a.name = "Alex"

# створимо якийсь блок кода
def code(obj):
    print(obj.name)

# прив'яжемо блок до класа як атрибут show_name
Users.show_name = code

# виконаємо із класа, якому блок кода належить прямо
Users.show_name(Users) # NoName

# виконаємо із інстанса, який унаслідував
a.show_name() # Alex

В останньому рядку метод не має параметра в дужках, хоча це вимагається. Це тому що перший параметр у нього автоматично додається - це посилання на інстанс.

Коли ми пишемо блоки кода в класі з прицілом, що їх будуть виконувати інстанси, то ми зобов'язані як мінімум мати один параметр у дужках. Є домовленість його називати змінною self. Далі в тілі блока кода ми маємо доступ до інстанса завдяки цій змінній.

def code(self):
    print(self.name)

Замість self буде підставлений конкретний інстанс.
